# PageMind — AI Page Summarization Chrome Extension

A Chrome extension that uses Google Gemini AI to summarize web pages and answer questions about their content.

## Features

- One-click page summarization
- Chat interface to ask questions about the current page
- Voice input support
- Draggable overlay UI

## Tech Stack

- JavaScript, Chrome Extension API (Manifest V3)
- Google Gemini AI API

## Installation

1. Clone this repo
2. Open Chrome → `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** → select this folder
5. Click the PageMind icon → enter your Gemini API key in settings

## Usage

- Click the orb to open the chat panel
- Type or speak your question about the current page
- Double-click the orb to get a full page summary
